University Thesis Project

Comparative Analysis of Instance Segmentation Models

This thesis presents a comprehensive comparative analysis of three state-of-the-art deep learning models for instance segmentation, specifically targeting tree canopy detection in high-resolution aerial imagery. The research evaluates the performance, scalability, and practical deployment potential of each model within real-world environmental monitoring workflows.

Research Methodology

  • Developed a custom, high-quality dataset of annotated tree instances from diverse aerial imagery sources.
  • Implemented and trained three leading instance segmentation architectures using PyTorch and TensorFlow, including Mask R-CNN, YOLACT, and YOLO-based segmentation.
  • Conducted rigorous quantitative and qualitative evaluation using metrics such as mAP, IoU, and inference speed.
  • Benchmarked model performance across varying dataset sizes and image resolutions to assess scalability and robustness.

Practical Deployment & GIS Integration

  • Integrated the top-performing model with QGIS, enabling seamless geospatial analysis and visualisation of segmentation results for end-users in forestry and environmental management.
  • Developed automated pipelines for data preprocessing, model inference, and georeferenced output generation.

Key Findings

  • Identified significant trade-offs between model accuracy, computational efficiency, and ease of deployment.
  • Demonstrated that custom dataset quality and annotation consistency are critical for high-precision segmentation in complex natural environments.
  • Showed that open-source deep learning frameworks (PyTorch, TensorFlow) enable rapid experimentation and reproducibility in applied remote sensing research.

Academic Contribution

This work advances the application of deep learning for environmental monitoring by providing a reproducible benchmark and practical deployment workflow for tree canopy instance segmentation. The integration with QGIS demonstrates real-world utility for researchers and practitioners in geospatial analysis.
